====== Enchanter ======
Created Saturday 13 January 2024

Each crafting class has a main and off hand tool.
[[AdvancedCrafting]] classes use a consumable item for the off hand.
Enchanters use an [[+EnchantingBook]] and an [[+Essense]] which is consumed on sucess or failure.
